=load-path= 上の全ファイルをシンボルプロパティインデックスとして事前構築することで、
=locate-library=、=require=、=load= を高速化します。

* インストール

** async-installer

[[https://github.com/zawatton21/async-installer][async-installer]] — Emacs 用の小型ノンブロッキングパッケージマネージャ。マクロ DSL 不要の純粋な Elisp 構文で使えます。

#+begin_src emacs-lisp
(async-installer-git-add "https://github.com/zawatton21/my-locate-library.git"
                            :branch "master"
                            :main "my-locate-library.el")
#+end_src

** straight.el

#+begin_src emacs-lisp
(straight-use-package
 '(my-locate-library :type git :host github :repo "zawatton21/my-locate-library"))
#+end_src

** elpaca

#+begin_src emacs-lisp
(elpaca (my-locate-library :host github :repo "zawatton21/my-locate-library"))
#+end_src

** use-package（straight または elpaca と組み合わせて）

#+begin_src emacs-lisp
;; straight と組み合わせる場合
(use-package my-locate-library
  :straight (my-locate-library :type git :host github :repo "zawatton21/my-locate-library"))

;; elpaca と組み合わせる場合
(use-package my-locate-library
  :ensure (:host github :repo "zawatton21/my-locate-library"))
#+end_src

** 手動インストール

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/my-locate-library")
(require 'my-locate-library)
#+end_src

* 使い方

#+begin_src emacs-lisp
;; load-path が確定した後に実行:
(my-locate-library-build)
(my-locate-library-enable)

;; 後から load-path が変更された場合:
(my-locate-library-rebuild)

;; 元の関数に戻す場合:
(my-locate-library-disable)
#+end_src
